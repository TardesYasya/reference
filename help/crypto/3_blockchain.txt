Блокчейн
  Блокчейн - способ организации данных, в котором данные практически невозможно сфабриковать.
  Блокчейн хранит данные в блоках, которые соединены между собой. Обычно данные в этих блоках - списки.
  Алгоритм конценсуса - алгоритм, говоряший, почему можно доверять этому блоку.

Алгоритм конценсуса PoW (Proof of Work)
  случайные данные, подбираемые майнерами для того, что-бы совершить какую-либо работу
  в данном случае подбирается данные для того что-бы хеш всего этого блока начинался с
  определённого количества нулей, после этого блок отправляется в блокчейн


Алгоритм конценсуса PoS (Proof of Stake)
  Узел (комп) который генерирует блок выбирается случайно, в процентном соотношении его ставки 
  к общиму числу монет. ставка - замороженные монеты, которые можно потерять
  при попытке фальсификации блока сгорает вся ставка или её часть.


Пример блока:
  +--------+---------------------+         +--------+---------------------+
  | prev   | 0000000000000000000 |    +--> | prev   | 00000082b43faddc378 |
  +--------+---------------------+    |    +--------+---------------------+
  | id     |                0001 |    |    | id     |                0002 |
  +--------+---------------------+    |    +--------+---------------------+
  | data   |               [...] |    |    | data   |               [...] |
  +--------+---------------------+    |    +--------+---------------------+
  | reward |                  50 |    |    | reward |                  50 |
  +--------+---------------------+    |    +--------+---------------------+
  | nonce  | 0x6f768b670082b2340 |    |    | nonce  | 0x8c594841a63492fba |
  +--------+---------------------+    |    +--------+---------------------+
  | hash   | 00000082b43faddc378 | ---+    | hash   | 00000af79b7970cc89f |
  +--------+---------------------+         +--------+---------------------+

Описание:
  +--------+--------------------------------------------------------------------------------------------+
  | prev   | хеш прошлого блока                                                                         |
  +--------+--------------------------------------------------------------------------------------------+
  +--------+--------------------------------------------------------------------------------------------+
  | id     | номер блока                                                                                |
  +--------+--------------------------------------------------------------------------------------------+
  +--------+--------------------------------------------------------------------------------------------+
  |        | +-------+-------+-------+-----------------------+-----+                                    |
  |        | | from  | to    | count | sign                  | fee |                                    |
  |        | +-------+-------+-------+-----------------------+-----+                                    |
  |        | | user1 | user2 |    30 | 1e4d0eacc41ab503e588c |   1 |                                    |
  |        | +-------+-------+-------+-----------------------+-----+                                    |
  |        | | user4 | user5 |    54 | ef45e8d14003678f746fd |   1 |                                    |
  | data   | +-------+-------+-------+-----------------------+-----+                                    |
  |        | | user1 | user6 |    12 | 8a69006270decec2b19b8 |   2 |                                    |
  |        | +-------+-------+-------+-----------------------+-----+                                    |
  |        | | user2 | user1 |     7 | 6a2a2061e78826bfb5e83 |   1 |                                    |
  |        | +-------+-------+-------+-----------------------+-----+                                    |
  |        | | user1 | user6 |    88 | 98f09604a21215f347070 |   3 |                                    |
  |        | +-------+-------+-------+-----------------------+-----+                                    |
  +--------+--------------------------------------------------------------------------------------------+
  +--------+--------------------------------------------------------------------------------------------+
  | reward | награда майнеру                                                                            |
  +--------+--------------------------------------------------------------------------------------------+
  +--------+--------------------------------------------------------------------------------------------+
  | nonce  | данные майнера                                                                             |
  +--------+--------------------------------------------------------------------------------------------+
  +--------+--------------------------------------------------------------------------------------------+
  | hash   | хеш всех перечисленных до этого данных                                                     |
  +--------+--------------------------------------------------------------------------------------------+
